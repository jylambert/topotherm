Feature Overview
================

* Single time-step topology and piping optimization.

  - Multi time-step version includes operation with variable heating demands.
  - Greenfield optimization.
  - Existing network support in development.
  
* Forced and economic expansion of the district heating network to consumers.
* Supports all Pyomo solvers (note: output helper functions in ``utils.py`` may need adjustments).
* Plotting functions included.
